# Mountain Lion Habitat Analysis

Showcase application for the [ArcGIS Maps SDK for JavaScript](https://js.arcgis.com/) that analyzes mountain lion habitat factors in the Santa Monica Mountains and greater Los Angeles region.

## Prerequisites

- Node.js 20+
- npm 10+

## Development

1. Install dependencies.

```bash
npm ci
```

1. Start the local development server.

```bash
npm run dev
```

1. Create a production build.

```bash
npm run build
```

1. Preview the production build locally.

```bash
npm run preview
```

## Deployment

GitHub Pages deployment is configured with a workflow in `.github/workflows/pages-deploy.yml`.
